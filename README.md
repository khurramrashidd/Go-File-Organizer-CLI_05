# 📂 Go File Organizer CLI (Day 5)

A practical and efficient **CLI-based File Organizer built using Go (Golang)** to automatically sort files into categorized folders.

This tool helps organize messy directories by grouping files based on their extensions.

This project is part of my **Go learning journey (Day 5 Project)**.

---

## 🚀 Features

* Automatically organizes files into folders
* Categorizes files (Images, Documents, Videos, Others)
* Creates folders dynamically if not present
* Works on any directory path
* Fast and lightweight CLI tool

---

## 🛠 Built With

* **Go (Golang)**
* Standard Library (os, filepath, fmt)

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/go_cli_05_file_organizer.git
```

### 2. Navigate into the folder

```bash
cd go_cli_05_file_organizer
```

---

## ▶️ Usage

Run the application:

```bash
go run main.go
```

Then enter the folder path you want to organize.
